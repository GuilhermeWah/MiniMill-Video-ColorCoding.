# Images Directory

Place your Molycop logo file here as `molycop-logo.png`.

The logo should be:
- PNG format (or SVG for better quality)
- Transparent background (recommended)
- Appropriate size for display (recommended: 200-400px width)

The logo will be displayed:
- In the navigation bar (height: 40px)
- On the hero section of the homepage (max-height: 400px)

If you have the logo in a different format or name, you can:
1. Rename it to `molycop-logo.png`
2. Or update the image source paths in the HTML files
